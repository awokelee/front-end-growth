## Why I learn Vue.js

### 精简版

- 公司后台管理系统需要搭建 UI 组件库
- 上级领导提议用 React
- 对比 React 与 Vue， 我选择了 Vue

### 流水账

2016 年 9 月，公司后台管理系统需要新的视觉规范，提出了搭建 UI 组件库的需求
